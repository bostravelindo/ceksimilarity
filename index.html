⭐ Quality Assessment (Penilaian Kualitas)
Nilai kualitas setiap literatur yang lolos screening menggunakan 8 kriteria adaptasi JBI Critical Appraisal. Skala 0-2 per kriteria (total maks. 16). Threshold minimum: 8/16 (50%).
Rubrik Penilaian: 0 = Tidak memenuhi / Tidak ada informasi · 1 = Sebagian memenuhi / Informasi terbatas · 2 = Sepenuhnya memenuhi / Informasi jelas dan lengkap
Threshold: ≥ 8/16 untuk lolos ke Data Extraction
